ファイル
  • 画像：画像のアップロード、設定、削除を行います
  • ZIP読み込み：保存したZIPファイルを読み込ませます
  • 保存：ルームデータ全てをローカルに保存します
  • HTML：全てのチャットタブのログをタブごとにHTML形式で保存します
ルーム
  • ルーム設定：ルーム全体にかかわる権限・設定が行えます
  • プレイヤー情報：自分のプレイヤーの設定、他のプレイヤーの確認、削除が行えます
  • テーブル設定：ゲームテーブル・背景の設定が行えます
  • 音楽：BGM・SEの再生・停止・削除が行えます
  • ダイス一斉公開：伏せられたダイスを公開します。
  • ラウンドをリセット：ラウンドをリセットします。
機能
  • チャット画面：チャットウィンドウを新たに表示します
  • インベントリ：キャラクターインベントリを表示します
  • パレットバインダー：チャットパレットを表示します
  • カウンターリスト：ゲームキャラクターにカウンターを配布します
  • ダイスボット表：カスタムダイスボット表を設定します
  • カットイン：カットインを設定します
  • エフェクト：ゲームキャラクターに３Dエフェクトを表示します
表示
パネル設定
  • 全てのパネルを消去：チャットウィンドウを新たに表示します
視点設定(３Dモード限定)
  • 初期視点に戻す：視点を初期に戻します
  • 真上から見る：視点をテーブルの真上に移動します
立ち絵設定
  • 立ち絵表示設定：立ち絵の表示位置の確認、表示範囲の設定ができます
  • 立ち絵表示：立ち絵表示のON/OFFを切り替えます
  • ネームタグ表示：立ち絵表示時のネームタグのON/OFFを切り替えます
  • 透明化、自動退去：立ち絵が発言後、自動的に消滅するかどうかを切り替えます
  • 表示中の立ち絵全消去：表示中の立ち絵を全て消します
エフェクト設定
  • エフェクト表示：エフェクト表示のON/OFFを切り替えます
