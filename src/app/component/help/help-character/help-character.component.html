メモ(任意のキャラクターを右クリック→メモ)
通常のステータスとは別に、フリーテキスト形式でメモを記載できます。
メモはオーバービュー時にステータスと切り替えて表示可能です。
エフェクト(機能→エフェクト)
キャラクターに３Dエフェクトを表示することが可能です
表示方法はエフェクトビューを開き、表示したいエフェクトを選択後、
エフェクトを付与をクリックし付与したいキャラクターをクリック。
またはチャット末尾に@effect(エフェクト名)で表示されます
チャットからのリソース操作
チャットからリソースを操作することが可能です
書式:
:[ステータス名]c(計算式)
:[ステータス名]=(ダイスコマンド)
:[ステータス名]+(ダイスコマンド)
:[ステータス名]-(ダイスコマンド)

例)
:HP-2d6
C(計算式)と＝はステータスの値を出力結果で置き換えます。
＋とーはステータスの値に加算・減算します。

ステータス名はデータが数値と扱えるものであればなんでも指定可能です。
ダイスコマンドはbcdiceから出力される結果が数値であれば使用可能です。
各システム用のコマンドでも問題ありません。
(最終結果が数値である必要はありません。
末尾に 成功 や クリティカル とついていても、過程に数値が出力されていれば大丈夫です。)
