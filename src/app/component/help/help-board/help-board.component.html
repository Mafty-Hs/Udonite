掲示板(Message Board)
画面右上に常時表示されている情報共有機能です
新規作成・更新があると赤く点滅して更新を通知します
掲示板を開く
Message Boardと書かれている場所をクリックすると情報一覧が表示されます。
新規作成されたものはNEW、更新されたものはUPDATEと表示され、
公開範囲が制限されたものには鍵のマークがつきます。
開きたいものをクリックすると内容が表示されます。
掲示板を作成する
新規作成をクリックすると作成画面が開きます。
ここで掲示板の権限を全体に公開、全体に公開(編集不可)、公開範囲を制限から選択できます。
権限に関わらず、作成したプレイヤーは編集可能です。
画像を設定する
掲示板には画像を設定することが可能です。
画像を設定した場合、画像をクリックすることでモニタが許す限り原寸大で画像を表示します。
